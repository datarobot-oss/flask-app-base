Flask Application Template

DataRobot

Getting started

This template gives you a working Flask application connected to the DataRobot platform. The application token is injected automatically — no manual authentication needed.

Add your own routes, call the DataRobot API through the built-in proxy endpoints, and explore the full API surface via the Swagger docs below.

Endpoints

GET /health Application health check GET /api/me Current account info GET /api/projects List projects GET /api/deployments List deployments GET /api/use-cases List use cases DOCS /apidocs Interactive Swagger UI
